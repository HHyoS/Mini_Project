2022-08-13 카드 뒤집기ver 2.0 

추가된 기능 :

    1. 사용자가 UX 개선을 위해 게임창의 위치를 화면 정중앙에 배치
    2. 게임 판의 사이즈 증가 및 이후 게임판 크기 조절시 자동으로 이미지 사이즈가 조절 되도록 코드 구현
    3. 게임의 느낌을 살리기 위해 같은 쌍 선택시 +5, 실패시 -3점의 점수 할당
    4. 게임 종료 후 얻은 점수 alet 메세지로 사용자에게 보여줌
